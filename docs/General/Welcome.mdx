## Note

Vortexus.js is currently under development and nothing has been published to the NPM registry yet

## About

Vortexus.js is a complex discord app framework made to be simple out of the box for developers

## Vortexus.js Packages

- [vortexus.js](https://github.com/PAdventures/Vortexus.js/blob/main/packages/vortexus.js/README.md) - The main package found between you, the developer, and Vortexus.js's packages so that you can run a Discord app
- [@vortexus.js/core](https://github.com/PAdventures/Vortexus.js/blob/main/packages/core/README.md) - Extends the [Discord.js](https://github.com/discordjs/discord.js) module which contains our builders and managers
- [@vortexus.js/utility](https://github.com/PAdventures/Vortexus.js/blob/main/packages/utility/README.md) - Utilities that are used by many if not all Vortexus.js packages

*PS: Vortexus.js is a [Reciple](https://github.com/thenorthsolution/Reciple) and [Saphhire](https://github.com/sapphiredev/framework)* inspired framework
