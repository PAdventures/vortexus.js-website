This guide is made for to help newcomers and experts to have the best experience as possible when developing with the Vortexus.js framework.
Keeping this guide as up-to-date as possible is important to keep that goal, therefore adding more content is a top priority and even you can request for more content.

To request for more content for the **guide**, head over to the [issue tracker](https://github.com/PAdventures/Vortexus.js/issues)<ExternalLink /> &nbsp;and [create a new issue](https://github.com/PAdventures/Vortexus.js/issues/new/choose)<ExternalLink />.
Make sure you select the "Feature Request" template to follow the correct format for issues as we are strict about keeping everything as readable as possible.

:::tip
Remember that you are always free to [`fork the repository`](https://github.com/PAdventures/Vortexus.js/fork)<ExternalLink /> &nbsp;and [`make a pull request`](https://github.com/PAdventures/Vortexus.js/pulls)<ExternalLink /> &nbsp;if you want to add something to the guide yourself
:::
